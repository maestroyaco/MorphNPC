Cómo Crear un NPC (Skin)

Pronto tendremos mucho mas contenido, pero por ahora te dejaremos unos pasos basicos, para usar los packs MorhpNPC

Por ejemplo:

  • Agregar el Pack MorhpNPC a retroclient/clips/sprites (Solo los SWF)
  • Crear un nuevo campo en la Base de datos npc_template y en gfxID poner el #ID del skin que descargaste
  • Editar el lang en: data/lang/swf/npc_es_#.swf agregando el ID identificador del NPC
  • Haz lo mismo para cada Skin de NPC descargado en el Pack MorhpNPC.
  • Tambien puedes usar los MorhpNPC para cambiar tu apariencia con el comando morph + ID

Pasa por nuestro Discord de servidores privados, para encontrar soporte.
